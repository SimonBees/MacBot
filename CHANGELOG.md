# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-28

### Added
- Initial macOS AI desktop assistant
- Electron + React + TypeScript architecture
- Chat interface with user/AI message bubbles
- Global hotkey (Cmd+Shift+M) to show/hide window
- System notification integration
- Zustand state management for chat history
- Tailwind CSS styling
- macOS native title bar (hiddenInset style)

### Features
- Real-time chat interface
- User and AI message differentiation
- Loading indicator for AI responses
- Window controls (minimize, hide, close)
- Auto-scroll to latest message

### Technical
- Separate build processes for Electron and React
- TypeScript for type safety
- Responsive design
- GitHub Actions for automated builds
